# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- Integration with NVIDIA NIM (NVIDIA Infrastructure Microservices).
- Support for `openai/meta/llama-3.2-90b-vision-instruct` for advanced skin condition diagnosis.
- Support for `openai/nvidia/llama-3.1-nemotron-70b-instruct` for enhanced conversational AI and recommendations.

### Changed
- Upgraded AI flows to utilize NVIDIA NIM models for better performance and accuracy.
- Updated Genkit configuration to support multiple AI providers (Google AI and NVIDIA NIM).
- Improved environment variable management with support for `NVIDIA_API_KEY`.
