## 📣 Reporting a Vulnerability

If you discover a security vulnerability within GlowPilot, please report it by:

1. **Email:** Send a detailed report to **mulkymalikuldhaher@email.com**
2. **Do NOT** create a public GitHub issue for security vulnerabilities
3. Include the following in your report:
   - Type of vulnerability
   - Full path of the affected file(s)
   - Steps to reproduce
   - Potential impact
   - Any possible mitigation

We will acknowledge your report within 48 hours and provide a detailed response within 7 days.

## 🔐 Security Best Practices

- All facial/skin images are processed locally and never stored on servers
- Firebase Authentication with Google Login provides secure user authentication
- All AI interactions are processed through secure Google Genkit endpoints
- User data is handled anonymously and in compliance with privacy standards
