# GlowPilot Copilot 🚀

GlowPilot Copilot is a virtual AI dermatology platform that analyzes skin conditions (via chat, voice, or photo) and provides personalized skincare routine recommendations. It features a futuristic interface with glassmorphism and aurora animations, powered by state-of-the-art AI models.

## ✨ Core Features

- **🤖 AI Dermatologist**: Advanced AI Chat Dermatologist (Text & Voice) for diagnosis and personalized skincare routines, now powered by **NVIDIA NIM & Llama 3**.
- **🔍 Skin Diagnosis**: Multi-modal analysis using **Llama 3.2 90B Vision** to analyze skin photos and provide preliminary observations.
- **🛒 Smart Product Catalog**: Automatically identifies relevant products and generates affiliate links for easy purchasing.
- **🎙️ Voice-First Experience**: Integrated STT + TTS (Gemini TTS) for a hands-free, interactive experience.
- **📱 Futuristic UI**: Modern, responsive interface with glassmorphism, aurora effects, and dark mode support.
- **📊 Progress Tracking**: Set goals and track your skincare journey over time.

## 🛠️ Tech Stack

- **Frontend**: Next.js 15, Tailwind CSS 4, Lucide React, Shadcn UI
- **AI Backend**: Genkit, NVIDIA NIM (Llama 3.1/3.2), Google Gemini
- **Database**: Firebase (Firestore, Auth)
- **Deployment**: Firebase App Hosting

## 🚀 Getting Started

To run the project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/mulkymalikuldhrs/GlowPilot_2.0.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Set up environment variables:**
    Create a `.env` file based on `.env.example` and add your:
    - `GEMINI_API_KEY`
    - `NVIDIA_API_KEY`
    - Firebase configuration
4.  **Run the development server:**
    ```bash
    npm run dev
    ```
5.  **Open [http://localhost:3000](http://localhost:3000)** in your browser.
