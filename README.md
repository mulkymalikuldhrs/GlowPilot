# GlowPilot Copilot

GlowPilot Copilot is a virtual AI dermatology platform that analyzes skin conditions (via chat, voice, or photo) and provides personalized skincare routine recommendations. It features a futuristic interface with glassmorphism and aurora animations, and is integrated with affiliate links for product purchases.

## 🚀 Key Improvements & Tech Stack

- **Next.js 15**: Leveraging the latest features of Next.js for better performance and developer experience.
- **Genkit AI**: Unified AI framework for managing flows, prompts, and tool calls.
- **NVIDIA NIM Integration**: Powered by NVIDIA's high-performance inference microservices.
    - **Llama 3.1 Nemotron 70B**: For advanced reasoning and conversational diagnosis.
    - **Llama 3.2 90B Vision**: For multimodal skin analysis and photo diagnosis.
- **Firebase**: Backend services for authentication and real-time database.
- **Tailwind CSS 4**: Modern styling with improved performance and developer workflow.

## Core Features

- **AI Dermatologist**: AI Chat Dermatologist (Text & Voice) for diagnosis and personalized skincare routines.
- **Product Scraper**: Automatically generates structured product data with affiliate links.
- **Voice Chat**: Integrated STT + TTS (Google AI Gemini TTS) for a hands-free experience.
- **Responsive UI**: Light/Dark mode and language support.
- **Product Comparison**: Compare products by price, rating, and ingredients.
- **Progress Tracking**: Track your skincare routine progress and set personalized goals.

## Getting Started

To run the project locally:

1.  Clone the repository:
    ```bash
    git clone https://github.com/mulkymalikuldhrs/GlowPilot_2.0.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Set up environment variables:
    Create a `.env` file with your `GEMINI_API_KEY`, `NVIDIA_API_KEY`, and Firebase credentials.
4.  Run the development server:
    ```bash
    npm run dev
    ```
5.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
